# AI-Based-Resume-Screening-and-Parsing-System
This project focuses on developing an AI-based system for resume screening and parsing. The system automatically extracts important information from resumes such as candidate name, skills, education, and experience using natural language processing techniques.

Overview
This project is designed to automate resume screening by extracting and analyzing candidate information using artificial intelligence. It uses natural language processing (NLP) techniques to parse resumes and identify key details, which are then used to evaluate candidate suitability.

Objectives

Automate resume parsing and information extraction
Reduce manual screening effort
Improve recruitment efficiency
Enable data-driven hiring decisions

Features

Resume text extraction
Information parsing (name, skills, education, experience)
Data cleaning and preprocessing
Candidate analysis and filtering
Machine learning/NLP-based processing

Technologies Used

Python
Pandas
NumPy
Natural Language Processing (NLP)
Scikit-learn
Jupyter Notebook
Regular Expressions (Regex)

Dataset
The dataset consists of resumes in text or PDF format containing candidate details such as:

Personal information
Skills
Educational background
Work experience

How It Works

Load resume data (PDF/text format)
Extract text from resumes
Clean and preprocess the text
Apply NLP techniques to extract key information
Analyze and categorize candidates

How to Run

pip install pandas numpy scikit-learn nltk spacy

View extracted information and results
